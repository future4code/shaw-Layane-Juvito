Projeto Loja Virtual

A ideia desse projeto é criar uma loja virtual, a príncipio não terá uma funcionalidade prática, é apenas para testar as habilidades de HTML e CSS puro. O site possui algumas páginas como:

  • Controle de Estoque
  • Venda de Produtos
  • Adição de novos produtos
  • Histórico de Vendas
